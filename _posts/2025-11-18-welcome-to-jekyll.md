# Jekyll 블로그 시작하기

안녕하세요! 드디어 Jekyll로 만든 GitHub Pages 블로그를 시작하게 되었습니다.

## Jekyll이란?

Jekyll은 정적 사이트 생성기(Static Site Generator)로, Markdown으로 작성한 글을 HTML로 변환해주는 도구입니다. GitHub Pages에서 공식적으로 지원하기 때문에 무료 호스팅이 가능합니다.

## Jekyll의 장점

1. **무료 호스팅** - GitHub Pages를 통해 무료로 호스팅할 수 있습니다
2. **간편한 글쓰기** - Markdown 문법으로 쉽게 글을 작성할 수 있습니다
3. **빠른 속도** - 정적 파일로 제공되어 로딩 속도가 빠릅니다
4. **커스터마이징** - HTML, CSS, JavaScript로 자유롭게 디자인할 수 있습니다

## 앞으로의 계획

앞으로 이 블로그에서는 다음과 같은 내용을 다룰 예정입니다:

- 프로그래밍 학습 내용
- 개발 과정에서 배운 것들
- 기술 관련 생각들
